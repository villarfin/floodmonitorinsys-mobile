# Flood Monitoring System Mobile

Separate mobile project for Lab 6 UI adaptation, mirroring the existing web system screens and logic.

## Screen Mapping

- `AdminLogin`
- `Dashboard`
- `Admin`
- `UserManagement`
- `Monitoring`
- `IncidentReport`
- `Notifications`
- `Summary`
- `Configuration`

## Reusable Components

- `MobileCard`
- `StatusBadge`
- `StatsCard`
- `WaterLevelCard`
- `AlertCard`

## Run

```bash
npm install
npm start
```

Open in Expo Go, Android emulator, or iOS simulator.
